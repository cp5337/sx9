# V0 Framework Configuration for SX9-Forge

## CRITICAL: Always include this preamble when using v0

When generating components, you MUST follow these constraints:

### Stack Requirements
- **Framework**: Vite 6 + React 18 (NOT Next.js, NOT Create React App)
- **Desktop**: Tauri 2.x (Rust backend, NOT Electron)
- **Styling**: Tailwind CSS 4 with @tailwindcss/postcss
- **Icons**: lucide-react (already installed)
- **State**: Redux Toolkit + Zustand (already configured)
- **Cross-platform**: react-native-web for View/Text/StyleSheet

### DO NOT USE
- ❌ `"use client"` or `"use server"` directives (these are Next.js only)
- ❌ `next/image`, `next/link`, `next/router` (use react-router-dom if needed)
- ❌ `getServerSideProps`, `getStaticProps` (no SSR)
- ❌ App Router patterns (`page.tsx`, `layout.tsx`, `loading.tsx`)
- ❌ Server Components or Server Actions
- ❌ `@vercel/analytics` or Vercel-specific packages

### DO USE
- ✅ Standard React functional components with hooks
- ✅ Inline styles with `React.CSSProperties` OR Tailwind classes
- ✅ `lucide-react` for icons: `import { IconName } from "lucide-react"`
- ✅ `@tauri-apps/api` for native features (filesystem, dialog, clipboard)
- ✅ `react-native-web` components when cross-platform is needed

### Import Patterns
```tsx
// Icons
import { Copy, Play, Settings } from "lucide-react";

// Tauri (when native features needed)
import { invoke } from "@tauri-apps/api/core";

// React Native Web (optional, for cross-platform)
import { View, Text, StyleSheet } from "react-native-web";

// Local aliases
import { Button } from "@/components/ui/button";
```

### Component Template
```tsx
import React from "react";

interface MyComponentProps {
  title: string;
  onAction?: () => void;
}

export const MyComponent: React.FC<MyComponentProps> = ({ title, onAction }) => {
  return (
    <div className="p-4 bg-slate-900 rounded-lg">
      <h2 className="text-lg font-semibold text-white">{title}</h2>
      <button
        onClick={onAction}
        className="mt-2 px-4 py-2 bg-cyan-500 text-white rounded"
      >
        Click me
      </button>
    </div>
  );
};

export default MyComponent;
```
